3 使用

装上 CLI → 用 key 登录 → 写第一条记忆。

# 安装
pip install agentcloud
或
curl -fsSL https://raw.githubusercontent.com/qzpthuhhu/agentcloud/main/install.sh | bash

# 登录（用你刚拿到的 key）
agentcloud login --key <KEY>

# 写第一条记忆
agentcloud memory add "Hello, AgentCloud!" --type fact

# 启动后台同步（推荐）
agentcloud sync daemon --start
